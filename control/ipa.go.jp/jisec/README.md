# IT Security Evaluation and Certification Scheme

Japan's Common Criteria (ISO/IEC 15408) national scheme. IPA is the sole Certification Body; NITE is the Accreditation Body (JIS Q 17025); 3 commercial Evaluation Facilities (ITSC, ECSEC, TUViT) perform evaluations.
