# PrivacyMark System

Operating since April 1, 1998. Based on JIS Q 15001:2017 (Personal Information Protection Management System - Requirements). Certification issued via JIPDEC or 19 designated assessment bodies; ~17,700+ certified entities as of mid-2026.

- **SecID:** `secid:control/jipdec.or.jp/privacymark`
- **Source:** https://privacymark.org/about/outline_and_purpose.html
- **License:** LicenseRef-Unclear-Quasi-Public-Body (unclear) -- see `privacymark-outline-metadata.json`

**Status:** Web page content fully captured; no separate downloadable standard document (JIS Q 15001 itself is a paid JSA standard, out of scope here).
