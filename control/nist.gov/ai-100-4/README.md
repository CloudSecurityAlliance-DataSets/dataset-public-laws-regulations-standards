# Adversarial Machine Learning: A Taxonomy and Terminology of Attacks and Mitigations

NIST AI 100-4 (2024) extends the AI 100-2 adversarial ML taxonomy to cover generative AI systems specifically. Defines attack categories (evasion, poisoning, privacy, abuse) and mitigation patterns for GenAI.
