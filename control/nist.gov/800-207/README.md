# Zero Trust Architecture

NIST Special Publication 800-207 defines Zero Trust Architecture (ZTA) — a security model centered on resources rather than network segments, with continuous authentication and authorization, least-privilege per-request access, dynamic policy enforcement, and assumption-of-breach posture.

- **Acronym:** SP 800-207
- **Owner:** National Institute of Standards and Technology (NIST)
- **SecID:** `secid:control/nist.gov/800-207`
- **Source:** https://csrc.nist.gov/pubs/sp/800/207/final
