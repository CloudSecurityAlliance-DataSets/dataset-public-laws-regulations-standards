# Enhanced Security Requirements for Protecting CUI

NIST SP 800-172 provides enhanced security requirements supplementing 800-171 — for use when CUI is high-value information or when nonfederal organizations face advanced persistent threats. Often required by DoD CMMC Level 3.

- **Acronym:** SP 800-172
- **Owner:** National Institute of Standards and Technology (NIST)
- **SecID:** `secid:control/nist.gov/800-172`
