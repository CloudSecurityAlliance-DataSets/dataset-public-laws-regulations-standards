# Module-Lattice-Based Key-Encapsulation Mechanism Standard

FIPS 203 (2024) standardizes ML-KEM (Module-Lattice-Based Key Encapsulation Mechanism, based on CRYSTALS-Kyber) — the first NIST post-quantum cryptography standard for key encapsulation. Replaces classical key-establishment mechanisms vulnerable to quantum computers.

- **Acronym:** FIPS 203
- **Owner:** National Institute of Standards and Technology (NIST)
- **SecID:** `secid:control/nist.gov/fips-203`
- **Source:** https://csrc.nist.gov/pubs/fips/203/final
