# A5 — AI Audit and Assurance Assessment Architecture

BSI's modular, expandable audit architecture for demonstrating the trustworthiness of AI systems, targeting all actors along the AI value chain (providers, operators, and those overseeing AI). It provides both criteria and assessment methodology, is structured per BSI's state-of-the-art library, and is published machine-readable in OSCAL. Broader in scope than AIC4, which is limited to AI cloud services.
