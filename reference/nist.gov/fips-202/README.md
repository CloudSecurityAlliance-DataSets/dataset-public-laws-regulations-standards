# SHA-3 Standard: Permutation-Based Hash and Extendable-Output Functions

FIPS 202 (2015) standardizes SHA-3 — the Keccak-derived hash family chosen via NIST's open competition. Complements (does not replace) SHA-2.

- **Acronym:** FIPS 202
- **Owner:** National Institute of Standards and Technology (NIST)
- **SecID:** `secid:control/nist.gov/fips-202`
- **Source:** https://csrc.nist.gov/pubs/fips/202/final
