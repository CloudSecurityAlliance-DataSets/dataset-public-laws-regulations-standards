# Stateless Hash-Based Digital Signature Standard

FIPS 205 (2024) standardizes SLH-DSA (Stateless Hash-Based Digital Signature Algorithm, based on SPHINCS+) — NIST's hash-based PQC signature standard, designed for higher security margin via different mathematical foundation than ML-DSA.

- **Acronym:** FIPS 205
- **Owner:** National Institute of Standards and Technology (NIST)
- **SecID:** `secid:control/nist.gov/fips-205`
- **Source:** https://csrc.nist.gov/pubs/fips/205/final
