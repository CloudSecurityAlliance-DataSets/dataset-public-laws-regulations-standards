# CISA Known Exploited Vulnerabilities Catalog

CISA's Known Exploited Vulnerabilities (KEV) Catalog — list of vulnerabilities known to be exploited in the wild. Federal agencies must remediate KEV entries by binding operational directive (BOD 22-01).

- **Acronym:** KEV
- **Owner:** Cybersecurity and Infrastructure Security Agency (CISA)
- **SecID:** `secid:reference/cisa.gov/kev`
- **Source:** https://www.cisa.gov/known-exploited-vulnerabilities-catalog
