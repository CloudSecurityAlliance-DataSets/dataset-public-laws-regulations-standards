## access control

A way of ensuring that only authorised users (or automated systems) can access data or services.

*see also: authentication*

## account freezing

Temporarily suspending an account, in response to the detection of unusual/suspicious activity.

*see also: throttling*

## active cyber defence

A set of NCSC services that are designed to reduce high-volume commodity cyber attacks.

*also known as: acd; mail check; web check | see also: acd; mail check; web check*

## administrator

A person who manages a computer system, usually with highly privileged access to data and services.

*also known as: admin*

## ai

Artificial intelligence (AI) describes computer systems which can perform tasks usually requiring human intelligence. This could include visual perception, speech recognition or translation between languages.

*also known as: artificial intelligence; machine learning | see also: artificial intelligence; machine learning*

## allow list

A security strategy organisations can use that approves a list of email addresses, IP addresses, domain names or applications, while denying all others, to protect computers and networks from threats.

*also known as: whitelist whitelisting and blacklisting is still used across the industry; but the ncsc do not use these terms. read more why in our blog post: /blog-post/terminology-its-not-black-and-white; deny list | see also: whitelist whitelisting and blacklisting is still used across the industry; but the ncsc do not use these terms. read more why in our blog post: /blog-post/terminology-its-not-black-and-white; deny list*

## android

Google's mobile operating system, used by several smartphone and tablet manufacturers.

*see also: operating system*

## anti-malware

Software that is designed to detect, stop and remove malicious software (malware).

*see also: antivirus*

## antivirus

Software that is designed to detect, stop and remove viruses and other kinds of malicious software.

*also known as: av; anti-malware | see also: av; anti-malware*

## application

A software program running on a computer. Usually referred to as an 'app' when running on a smartphone or tablet.

*also known as: app*

## apt

Advanced persistent threat. A nation-state or highly capable criminal actor that gains unauthorised access to a system and remains undetected for a long time.

*also known as: advanced persistent threat; hacker | see also: advanced persistent threat; hacker*

## artificial intelligence

Artificial intelligence (AI) describes computer systems which can perform tasks usually requiring human intelligence. This could include visual perception, speech recognition or translation between languages.

*also known as: ai; machine intelligence; machine learning | see also: ai; machine intelligence; machine learning*

## asset management

Identifying and recording of an organisation's physical assets, software, data, essential staff and utilities.

*see also: configuration management*

## attack tree

A systematic method for analysing and depicting the different ways a computer system might be attacked.

*see also: hacker*

## attack vector

The method an attacker uses to gain unauthorised access to a computer system or device.

*see also: hacker*

## attacker

Individual or group that conduct cyber attacks.

*also known as: threat actor; cyber attack | see also: threat actor; cyber attack*

## authentication

The process of determining if someone (or something) is who (or what) it claims to be.

*see also: credentials; access control*

## autorun

An option in Windows to automatically run programs contained on removable media when inserted into the device.

*see also: removable media*

## backup

A copy of computer data, usually stored in the cloud or on removable media, that can be used to recover from data loss or theft.

## bgp

Border Gateway Protocol. A protocol that exchanges routing information across the internet, typically between ISPs or large organisations.

*also known as: border gateway protocol; isp | see also: border gateway protocol; isp*

## biometrics

A type of authentication based on biological data that is unique and specific to a person (such as a fingerprint or face ID).

*see also: authentication*

## bios

Basic input-output system. The firmware built into PCs that connect the vital components, now superseded by UEFI.

*also known as: basic input-output system; uefi | see also: basic input-output system; uefi*

## bluetooth

A short-range, wireless means of communicating between devices (mobile phones, tablets, computers).

*see also: wifi*

## botnet

A distributed network of infected devices on the internet, used to commit co-ordinated cyber attacks without their owner's knowledge.

*see also: ddos*

## breach

An incident in which data, computer systems or networks are accessed or affected without authorisation.

*also known as: data breach*

## browser

An application that displays information and accesses services from the web. Edge, Safari, Firefox and Chrome are all browsers.

*also known as: web browser; www | see also: web browser; www*

## brute force attack

An attack that uses computing power to automatically enter a large number of guesses for security credentials in an attempt to gain unauthorised access.

*see also: credentials*

## bulk data

Term used to describe large quantities of (often personal) data that needs special protection.

*see also: encryption*

## byod

Bring your own device. When employees use their own device(s) for work purposes, but the employer owns corporate data and resources that may be accessed or stored on it.

*also known as: bring your own device*

## ca

Certificate authority. An organisation that issues digital certificates to allow the authentication and secure exchange of data

*also known as: certificate authority*

## carrier

An organisation that offers communication services over land-wire, sea cable, mobile, and/or satellite systems.

*also known as: isp | see also: isp*

## cas

Competent authorities. The organisations responsible for enforcing regulation under the NIS Directive.

*also known as: competent authorities*

## certificate

A form of digital identity for a computer, user or organisation to allow the authentication and secure exchange of data.

*see also: ca*

## certification

A form of digital identity for a computer, user or organisation to allow the authentication and secure exchange of data.

## chatgpt

An artificial intelligence language model developed by OpenAI designed to mimic human conversation, based on the input it receives.

*see also: ai; large language model; llm; artificial intelligence*

## cloud

An on-demand, massively scalable service, hosted on shared infrastructure, accessible via the internet. Typical services include providing data storage, data processing, and pre-built functionality, such as logging.

## cms

Content Management System. Software used to create, manage and publish digital content, usually to the web.

*also known as: content management system*

## configuration management

Processes for defining and maintaining the consistency of configurations of software, hardware and other elements of an organisation to ensure reliable performance through its life.

*see also: asset management*

## credential stuffing

A method where attackers use passwords stolen from one web service to try and access accounts on another service, taking advantage of any reuse of username and password combinations.

*see also: credentials; passwords*

## credentials

Information used to verify someone's identity, such as a password, token or certificate, that authenticates use of a service.

*see also: passwords*

## critical national infrastructure (cni)

National assets that are essential for the functioning of society, such as those associated with energy supply, water supply, transportation, health and telecommunications.

*also known as: cni*

## cryptography

Technologies primarily based on maths and computer science for protecting information so that it is only accessible to authorised recipients.

*see also: encryption*

## csirt

Cyber security incident response team. An expert group that manages the response to a cyber security event or incident.

*also known as: computer security incident response team*

## cyber assessment framework

A tool for assessing how well suppliers of essential services are managing cyber security risks under the NIS Directive.

*also known as: caf*

## cyber attack

An attempt to damage, disrupt or gain unauthorised access to computer systems, networks or devices.

*also known as: hack incident; attacker; cyber threat; hacker; insider threat | see also: hack incident; attacker; cyber threat; hacker; insider threat*

## cyber essentials

A government-backed, industry supported scheme to help organisations protect themselves against common cyber attacks.

*see also: cyber attack*

## cyber security

The protection of devices, services and networks - and the information on them - from unauthorised access, theft or damage.

*see also: ncsc*

## cyber strategy

A long-term plan of action with the aim of implementing cyber security.

*see also: ncsc*

## cyber threat

The threat of a cyber attack to a user or organisation, and the unauthorised access, theft or damage that could result.

*see also: hacker; cyber attack*

## cyberfirst

Scheme run by the NCSC that encourage children and young adults to consider a career in cyber security.

## data at rest

Data in a persistent state such as when stored on hard disks, removable media or backups.

*see also: removable media*

## data centre

A managed facility that houses data storage and computer systems.

## data in transit

Data as it is transferred from one location to another, either through a private network or the internet.

*see also: data at rest*

## ddos

Distributed Denial of Service. A type of cyber attack where a computer service is overloaded by requests from multiple computers or locations, so that real users can no longer access the service.

*also known as: distributed denial of service; botnet; dos | see also: distributed denial of service; botnet; dos*

## decryption

Making information that has been encrypted readable again.

*see also: cryptography*

## deny list

A security control that organisations can put in place which blocks their users from connecting to or accessing named entities (such as IP addresses or domains) that may be suspicious.

*also known as: blacklist whitelisting and blacklisting is still used across the industry; but the ncsc do not use these terms. read more about the decision in the blog post - &#039;terminology: it&#039;s not all black and white&#039;; allow list | see also: blacklist whitelisting and blacklisting is still used across the industry; but the ncsc do not use these terms. read more about the decision in the blog post - &#039;terminology: it&#039;s not all black and white&#039;; allow list*

## devices

Computer-based hardware that physically exists, such as a desktop computer, smartphone or tablet.

## dictionary attack

A type of brute-force attack where words in a dictionary, phrases or common passwords are used to guess credentials.

*see also: brute force attack*

## digital footprint

The traces of digital information or 'footprint' that a person's online activity leaves behind.

## digital services

The provision of services (such as banking or online shopping) via the internet, or an electronic network.

## digital signature

A technique used to validate the authenticity and integrity of a message, software or digital document.

## dkim

DomainKeys Identified Mail. An email authentication method that adds a digital signature to every outgoing message, to verify the origin of the message.

*see also: mail check*

## dma

Direct Memory Access. Allows devices to access main memory outside of the control of the CPU, allowing faster data rates

## dmarc

Domain-based Messaging and Reporting Conformance. An email authentication and reporting protocol that helps to monitor and improve protection from fraudulent email.

*see also: mail check*

## dmz

Demilitarised zone. A controlled segment of an organisation's private network (which typically separates it from the internet), used to host their publicly accessible services such as websites.

*also known as: demilitarised zone; firewall | see also: demilitarised zone; firewall*

## dns

Domain Name System. The directory service that translates internet domain names (for example www.bbc.co.uk) to their corresponding IP addresses.

*also known as: domain name service; ip address | see also: domain name service; ip address*

## domain

A set of network addresses on the internet, managed by an organisation or person.

## dos

Denial of Service. A type of cyber attack where a computer service is overloaded, so that real users can no longer access the service.

*also known as: denial of service; ddos | see also: denial of service; ddos*

## download attack

A type of cyber attack where the victim unintentionally installs malware (such as viruses) on to a device, without the owner's knowledge.

*also known as: drive-by download; malware | see also: drive-by download; malware*

## education

The NCSC provides cyber security guidance and resources for the education sector to help protect everyone from schools to universities.

*also known as: research &amp; academia; research & academia | see also: research &amp; academia; research & academia*

## encryption

Protection of information by making it unreadable by everyone except those with the key to decrypt it.

*see also: cryptography*

## end user device

Term to collectively describe modern smartphones, laptops and tablets that connect to an organisation's network.

*also known as: eud; mdm | see also: eud; mdm*

## enterprise

Any large business or organisation that uses computer systems, networks and devices.

## equality, diversity and inclusion

Equality, Diversity and Inclusion

## escrow

A trusted third party service that holds money, software or other assets pending completion of a transaction.

*see also: asset management*

## ethernet

The networking technology that uses wires (rather than Wi-Fi) to connect devices to networks (including the internet).

*see also: lan*

## exercising

Cyber incident exercising helps organisations to establish how resilient they are to cyber attack, and practice their response in a safe environment.

## exploit

Development of tools to take advantage of a vulnerability in a system to gain unauthorised access or produce unwanted effects

*see also: attacker; vulnerabilities*

## fido2

Fast IDentity Online 2 is a set of standards that enable authentication using cryptographic keys instead of passwords.

*see also: authentication*

## fingerprint reader

A biometric device that scans fingerprints to authenticate users.

*see also: biometrics*

## firewall

Hardware or software used to prevent unauthorised access to or from a network.

*see also: network*

## firmware

The software embedded into a device that allows it to function.

*see also: bios; uefi*

## gateway

Networking appliance that allows data to flow between different types of networks.

*see also: router*

## hacker

Someone who uses their technology skills to break into computers, systems and networks.

*see also: attacker*

## hashing

Converting data into a fixed-length unreadable format that can't easily be reversed. Commonly used to protect passwords.

*see also: cryptography; encryption; salt*

## honeypot

Infrastructure (such as a web server) designed to attract attackers in order to learn how better to defend systems.

*see also: attacker*

## host

A network endpoint. For example, a PC, laptop, smartphone or tablet.

## hotspot

The physical location within the range of a wifi access point that provides access to the internet.

*see also: wifi*

## hypervisor

Computer software or hardware that can run and manage one or more virtual machines.

## ics

Industrial Control System. Systems, often involved with critical national infrastructure, which control and respond to physical feedback.

*also known as: industrial control system; critical national infrastructure (cni) | see also: industrial control system; critical national infrastructure (cni)*

## identity provider

An organisation (such as Google, Microsoft and Facebook) that provides online user identification on behalf of third parties.

## incident management

Activities to identify, analyse, and determine the response to cyber security incidents, to minimise the immediate and long-term business impact.

*see also: operational security*

## information assurance

Managing the risks to data, networks and information systems

*also known as: ia; risk management | see also: ia; risk management*

## infrastructure as a service (iaas)

A business model where consumers access virtualised hardware and software resources over the internet.

*also known as: iaas; platform as a service (paas); software as a service (saas) | see also: iaas; platform as a service (paas); software as a service (saas)*

## insider threat

Deliberate or accidental threat to an organisation's security from someone who has authorised access (such as an employee).

*see also: attacker*

## internet

The global 'network of networks' that provides a vast range of services such as the world wide web, email, social media and streaming services.

*see also: www*

## internet of things

Technology describing everyday objects (rather than computers and smartphones) that can connect to the internet. Examples include speakers, televisions and security cameras.

*also known as: iot*

## ip

Internet Protocol.  A communications protocol that relays data over network boundaries, and establishes routing across the internet.

*also known as: internet protocol; bgp; ip address; router; protocol | see also: internet protocol; bgp; ip address; router; protocol*

## ip address

Internet Protocol address. A number assigned to each device that is sending or receiving data across the internet or a LAN.

*see also: ip; router*

## ipsec

Internet Protocol Security. A suite of protocols that secures data sent over an IP network, often used to establish VPNs.

*also known as: internet protocol security; vpn | see also: internet protocol security; vpn*

## isp

Internet Service Provider. A company that provides individuals and organisations access to the internet, and other related services.

*also known as: internet service provider; carrier | see also: internet service provider; carrier*

## ixp

Internet Exchange Point. A location where ISPs exchange data between their networks.

*also known as: internet exchange point*

## keylogger

Hardware or software used to covertly record each keystroke entered into a computer or device.

*see also: end user device; keystroke*

## keystroke

A single press of a key on a computer.

## kms

Key management system. The processes used to generate, store, share, use and destroy cryptographic keys

## lan

Local area network. A collection of computers and devices in the same location (usually a building), connected by ethernet or wifi.

*also known as: local area network; wan; wifi | see also: local area network; wan; wifi*

## large language model

Large language models (LLMs) use algorithms trained on a huge amount of data, turning relationships between pieces of data into probabilities to predict sequences of text (or increasingly other content) in response to user prompts.

*also known as: llm; ai; artificial intelligence | see also: llm; ai; artificial intelligence*

## llm

Large language models (LLMs) use algorithms trained on a huge amount of data, turning relationships between pieces of data into probabilities to predict sequences of text (or increasingly other content) in response to user prompts.

*also known as: large language model; ai; artificial intelligence | see also: large language model; ai; artificial intelligence*

## logging

Generating, capturing, retaining and storing information on security-relevant events or behaviour from computer systems.

*also known as: security monitoring | see also: security monitoring*

## machine learning

Machine learning (ML) is a type of AI by which computers find patterns in data or solve problems automatically without having to be explicitly programmed. Most AI in current use is built using ML techniques.

*also known as: ml; ai | see also: ml; ai*

## macro

A small program that can automate tasks in applications (such as Microsoft 365) which attackers can exploit to cause harm.

*see also: vulnerabilities*

## mail check

An NCSC service that helps public sector organisations improve the security of their email domains.

*see also: active cyber defence*

## malvertising

Using online advertising as a delivery method for malware.

*also known as: malicious advertising; malware | see also: malicious advertising; malware*

## malware

Derived from 'malicious software', malware is any kind of software that can damage computer systems, networks or devices. Includes viruses, ransomware and trojans.

*also known as: malicious software; virus | see also: malicious software; virus*

## mdm

Mobile Device Management. Software that simplifies the administration and configuration of smartphones, laptops, tablets and other devices across an organisation.

*also known as: mobile device management; devices | see also: mobile device management; devices*

## metadata

Metadata describes the properties of data and communications. Examples include file date and packet size.

## mitigation

Actions that organisations and individuals can take to minimise and address risks.

## monitoring

The analysis of logging information to identify unusual events or patterns of behaviour.

*see also: logging*

## national protective security authority (npsa)

The National Protective Security Authority (NPSA) is the UK’s National Technical Authority for physical and personnel protective security.

*also known as: npsa*

## ncsc

The National Cyber Security Centre, the UK national technical authority for cyber security.

*also known as: national cyber security centre*

## ncsc for startups

An NCSC initiative where startup organisations work with NCSC experts to address important cyber challenges.

## network

Multiple computers and/or devices connected to shared resources and information.

*see also: lan; wan*

## oauth

A protocol that allows a 3rd party (for example, Google or Microsoft) to authorise access to services on a user's behalf.

*see also: authentication*

## oem

Original Equipment Manufacturer. An organisation that makes hardware or software that's used in other products.

*also known as: original equipment manufacturer*

## open source

When referring to software, software for which the source code is made freely available for use and modification.

## operating system

The basic software running on computers, tablets and smartphones, required to run additional applications and hardware.

*also known as: os; platform | see also: os; platform*

## operational security

The day-to-day protection of computer systems from cyber attack. Examples include backup, incident management and patching.

*see also: incident management; patching*

## operational technology

Technology that interfaces with the physical world and includes Industrial Control Systems (ICS), Supervisory Control and Data Acquisition (SCADA) and Distributed Control Systems (DCS)

*also known as: ot*

## passkeys

A more secure alternative to passwords that you don't need to remember as they are created and managed safely on your device(s).

*see also: authentication; passwords*

## passphrase

A sequence of words or other characters longer than a normal password, used to verify the identity of a user, usually to gain access to an account, website or system.

*see also: passwords*

## password manager

Software which can generate, store and retrieve passwords.

*see also: passwords*

## password spraying

Using a small number of commonly-used passwords in an attempt to access to a large number of accounts.

*see also: brute force attack*

## passwords

A sequence of characters used to verify the identity of a user, usually to gain access to an account, website or system.

*see also: passphrase*

## patching

Installing updates to devices or software to improve security and/or enhance functionality.

*see also: vulnerabilities*

## pattern

An architectural pattern for secure system design

*also known as: system design*

## penetration testing

An authorised test of a computer network or system, designed to look for security weaknesses with the end aim of fixing them.

*also known as: pen testing; pentest; vulnerabilities | see also: pen testing; pentest; vulnerabilities*

## pentest

Short for 'penetration testing,' an authorised test of a computer network or system, designed to look for security weaknesses with the end aim of fixing them.

*also known as: penetration test*

## people-centred security

Cyber security that focuses on the behavioural traits of the people who follow the processes put in place by organisations.

## personal data

Information that relates to an identified or identifiable individual, such as a name or number, or information such as an IP address.

## personnel security

Protecting organisations from the risks associated with the people it employs.

## pharming

A cyber attack where a person is misdirected to a fraudulent website, despite them entering the correct address.

## phishing

Scam emails or text messages that contain links to websites which may contain malware, or may trick users into revealing sensitive information (such as passwords) or transferring money.

*see also: smishing; spear-phishing; whaling; malware*

## physical security

Protecting the places where people work, and the locations of systems, services and networks.

## plain text

In cryptography, plain text refers to text that has not been encrypted (and is therefore easily readable).

*also known as: clear text*

## platform

Hardware or software environments on which applications can run. Software examples include Windows, Android and iOS. Hardware examples include PCs and Mac computers.

## platform as a service (paas)

A business model that allows developers to create and manage software applications, without having to own or maintain the physical infrastructure on which they run.

*see also: infrastructure as a service (iaas); software as a service (saas)*

## powershell

A powerful command-line tool and scripting language, often used to automate administrative and configuration tasks on Windows computers.

## protocol

A set of rules that determine how data is handled, protected and transmitted between devices on a network.

## ransomware

A type of malware which prevents you from accessing your device and the data stored on it, usually by encrypting your files. A criminal group will then demand a ransom in exchange for decryption.

*see also: malware*

## ransomware as a service

A business model where consumers purchase ransomware services 'off the shelf', without requiring the technical skills to develop the malware.

*also known as: raas; ransomware; malware | see also: raas; ransomware; malware*

## remote working

Working away from your business premises.

*also known as: home working; teleworking; byod | see also: home working; teleworking; byod*

## removable media

Devices such as USB flash drives, DVDs and memory cards, that can be easily removed from a computer to increase storage capacity (or to transfer data).

## research &amp; academia

Research & Academia and education

*also known as: education*

## rfc

Request for Comments. Detailed proposals concerning an aspect of internet technology, many of which become internet standards.

*also known as: request for comments*

## rfid

Radio frequency identification. Technology that uses electromagnetic fields to identify, track and share data between tags within objects or worn by people.

*also known as: radio frequency identification*

## risk

Possible future outcomes that we can describe in terms of their chances of occurrence, and the impact they would have if realised.

*see also: risk management*

## risk appetite

The level of risk that an organisation is prepared to take in pursuit of its objectives.

*see also: risk; risk management*

## risk management

The means of governing, communicating and making decisions about risk.

*see also: risk; risk appetite; risk owner*

## risk owner

A person who is accountable for risk within an organisation.

*see also: risk management*

## router

A device which sends data from one network to another based on the destination IP address.

*see also: gateway; ip address*

## rpz

Response policy zone. A filtering mechanism that prevents users from visiting harmful internet sites. Also known as a DNS firewall.

*also known as: response policy zone; dns | see also: response policy zone; dns*

## salt

Random data that is added to the password hashing process to help protect against dictionary attacks.

*see also: hashing*

## sanitisation

Techniques that ensure data stored on devices is removed so that it can no longer be recovered by unauthorised parties.

## secure by default

Technology in which security is embedded from its inception, and is frictionless for the user. Security is treated as a core fundamental rather than a follow-up activity.

## secure design and development

Activities relating to the design and development of IT systems and software that are secure by default.

## secure storage

Protecting digitally stored information using physical and electronic techniques.

## security architecture

The practice of designing the technology, people and processes within computer systems to achieve security goals.

## security monitoring

Active analysis by organisations of logging information to look for signs of attacks or unusual behaviour, to help detect events that could be considered an incident and to respond accordingly.

*also known as: protective monitoring; logging; soc; siem | see also: protective monitoring; logging; soc; siem*

## server

A software- or hardware-based device that accepts and responds to requests made over a network.

## shadow it

Refers to the unknown assets that are used within an organisation for business purposes.

*also known as: grey it*

## shoulder surfing

Observing information (such as a password) as it is being typed into a screen.

*see also: passwords*

## siem

Technology that supports threat detection and security management by collecting, monitoring and analysing security-related data from computer logs and alerts.

*also known as: security information and event management; logging | see also: security information and event management; logging*

## single sign-on

Using a single set of credentials (such as the same login and password combination) to access multiple services.

*also known as: sso; credentials | see also: sso; credentials*

## smishing

Combination of 'SMS' and 'Phishing'. Untargeted text messages sent to large numbers of people, asking for sensitive information, or encouraging them to visit a fake website.

*also known as: sms phishing; phishing | see also: sms phishing; phishing*

## soc

Security operations centre. A centralised facility within an organisation, responsible for activities such as security monitoring and incident management.

*also known as: security operations centre; incident management; security monitoring | see also: security operations centre; incident management; security monitoring*

## social engineering

A technique an attacker uses to manipulate people into carrying out specific actions, or divulging information.

*see also: phishing; smishing; spear-phishing; whaling*

## social media

Websites and apps, such as Facebook, X and Instragram, that allow people to share and respond to user-generated content (text posts, photos and video).

*see also: digital footprint*

## sociotechnical security

Security that accounts for the interaction of technology with people, processes and organisations.

## software as a service (saas)

A business model where customers access centrally hosted software applications over the internet (or network).

*see also: infrastructure as a service (iaas); platform as a service (paas)*

## spam

Unsolicited email sent in bulk, also referred to as junk email.

*also known as: junk email*

## spear-phishing

A form of phishing targeting particular individuals, where the email is designed to look like it's from a trusted or known person.

*also known as: whaling; phishing | see also: whaling; phishing*

## spoofing

A technique attackers use to appear to make their communications appear from a legitimate source. Email addresses, display names and domains can all be spoofed.

## spyware

A type of malware that installs on a device without the user's consent, collecting data and then sendng it to a third party.

*see also: malware*

## ss7

Signalling System 7. An international standard allowing phone networks to exchange the information needed for passing calls and text messages.

*also known as: signalling system 7*

## ssl

Secure Sockets Layer. An older method of encrypting data to communicate across the internet. Replaced by TLS.

*also known as: secure sockets layer; encryption; tls | see also: secure sockets layer; encryption; tls*

## supply chain

The extended network of trading relationships relied upon to deliver products, systems, and services.

## takedown

The removal of malware and phishing websites to limit the harm they can cause.

## tethering

Connecting to a phone using wifi, to share its internet connection.

*see also: wifi*

## threat intelligence

Information about threats that has been aggregated, analysed and enriched to provide the useful context for decision-making processes.

## throttling

Forcing a time delay between successive login attempts in order to defend against brute force attacks.

*see also: brute force attack*

## tls

Transport Layer Security. A cryptographic protocol that provides secure communications across computer networks, such as the internet.

*also known as: transport layer security; encryption | see also: transport layer security; encryption*

## token

A physical device or digital data, typically used with a password, to provide a user with access to a computer or network.

*see also: passwords*

## translation monitoring

A branch of security monitoring designed to protect online systems and services.

*see also: security monitoring*

## trojan

A type of malware, disguised as legitimate software, that is used to gain unauthorised access to a victim's computer.

*see also: virus*

## typosquatting

A technique used by threat actors where they set up a website or software with a misspelling of something legitimate. This can trick a user who accidentally misspells a service into entering information into the malicious website or downloading malware.

*also known as: typosquatting; attacker; hacker; cyber attack | see also: typosquatting; attacker; hacker; cyber attack*

## uefi

Unified Extensible Firmware Interface. A replacement for the BIOS (Basic Input-Output System) firmware that connects a PC's vital components.

*also known as: unified extensible firmware interface; bios; firmware | see also: unified extensible firmware interface; bios; firmware*

## uk gdpr

UK GDPR is the General Data Protection legislation in the UK that gives citizens more control over their personal data.

*also known as: general data protection regulation*

## video conferencing

A live audio and video conversation between two or more people, conducted using phone, tablet, laptop or desktop computer.

## virtual machine

A virtual environment within a physical computer that behaves like a separate computer. It allows the user to run programs and test apps in isolation.

## virtualisation

The process of creating a virtual machine.

## virus

A type of malware that is designed to infect legitimate software programs and replicates across networks when those programs are activated.

## virus signature

Files containing data unique to particular pieces of malware, including viruses, that security software downloads in order to aid detection.

## vpn

Virtual Private Network. A set of cryptographic technologies used to encrypt data as it travels over a network between two fixed endpoints.

## vulnerabilities

A weakness, or flaw, in software, a system or process. An attacker exploits these to (for example) gain unauthorised access to a computer system.

*see also: exploit*

## watering hole

A type of cyber attack where an attacker identifies a website frequented by a particular group of users, and then compromises that website with malware to infect the computers of its visitors.

*also known as: watering hole attack; download attack | see also: watering hole attack; download attack*

## worm

## 5g

5G is the fifth generation of wireless technology being rolled out in the UK, used to deliver the mobile internet without connecting to a cable or wifi. 5G is faster than the previous 3G and 4G generations, with reduced latency (the time between instructing a wireless device to perform an action and that action being completed) and greater capacity, allowing more devices in one area to be connected at the same time.

*see also: tethering*
