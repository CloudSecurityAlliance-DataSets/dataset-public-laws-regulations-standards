# Act on the Promotion of Ensuring National Security through Integrated Implementation of Economic Measures

Act No. 43 of Reiwa 4 (2022). Establishes 4 pillars: supply-chain resilience for specified critical products, critical-infrastructure operator designation and screening, support for critical/advanced technology development, and a non-publication system for sensitive patent applications.
