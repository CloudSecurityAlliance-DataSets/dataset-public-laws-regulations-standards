# Cyber Resilience Act

EU Cyber Resilience Act (Regulation (EU) 2024/2847) — horizontal cybersecurity requirements for products with digital elements placed on the EU market. Imposes secure-by-design, vulnerability disclosure, software supply chain, and CE-marking obligations on manufacturers.

- **Owner:** European Union
