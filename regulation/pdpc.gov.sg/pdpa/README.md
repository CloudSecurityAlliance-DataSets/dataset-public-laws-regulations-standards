# Personal Data Protection Act

Singapore's Personal Data Protection Act 2012 — governs the collection, use, and disclosure of personal data by organizations. Significantly amended in 2020 with mandatory breach notification.

- **Acronym:** PDPA (Singapore)
- **Owner:** Personal Data Protection Commission, Singapore
- **SecID:** `secid:regulation/pdpc.gov.sg/pdpa`
